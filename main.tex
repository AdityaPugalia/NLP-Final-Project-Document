\section{CORE Evaluation Suite}
\label{app:core}

We reproduce the task list from Appendix~G.I of the \textit{DataComp-LM} paper
\parencite{li2025datacomplmsearchgenerationtraining}, which defines the 22 tasks comprising the
CORE score.
The CORE score is the mean of centered accuracies on these tasks; centered accuracy is defined as
raw accuracy minus chance-level accuracy, so 0 corresponds to random guessing and 1 to perfect
performance.
These tasks were selected for their robustness to noise during training, even at small model
scales.

\begin{itemize}

  \item \textbf{AGI Eval LSAT-AR} \parencite{zhong2023agieval} (3-shot, 230 examples):
  Tests legal-domain knowledge and analytical reasoning.

  \item \textbf{ARC Easy \& ARC Challenge} \parencite{clark2018arc} (10-shot; Easy: 2,376,
  Challenge: 1,172 examples): Four-way multiple-choice questions from grade 3--9 science exams.
  Easy questions require basic science knowledge; challenge questions require procedural
  reasoning.

  \item \textbf{Big-Bench} \parencite{bigbench2023} (10-shot, 6 datasets):
  \begin{itemize}
    \item \textbf{QA Wikidata} (20,321 examples): Factual statement completion.
    \item \textbf{Dyck Languages} (1,000 examples): Completing balanced bracket expressions.
    \item \textbf{Operators} (210 examples): Computing outputs of newly defined operators.
    \item \textbf{Repeat Copy Logic} (32 examples): Instruction following with text operations.
    \item \textbf{CS Algorithms} (1,320 examples): Executing algorithmic reasoning tasks.
    \item \textbf{Language Identification} (10,000 examples): Identifying the language of text.
  \end{itemize}

  \item \textbf{BoolQ} \parencite{clark2019boolq} (10-shot, 3,270 examples):
  Binary question answering over relevant passages.

  \item \textbf{CommonsenseQA} \parencite{talmor2019commonsenseqa} (10-shot, 1,221 examples):
  Five-way multiple-choice commonsense QA.

  \item \textbf{COPA} \parencite{roemmele2011copa} (0-shot, 100 examples):
  Causal reasoning via plausible alternative selection.

  \item \textbf{CoQA} \parencite{reddy2019coqa} (0-shot, 6,304 examples):
  Conversational QA with answer extraction from passages.

  \item \textbf{HellaSwag} \parencite{zellers2019hellaswag} (0-shot and 10-shot, 10,042 examples):
  Four-way commonsense sentence completion.

  \item \textbf{Jeopardy} \parencite{kaggle2019jeopardy} (10-shot, 2,117 examples):
  Wide-coverage quiz questions in Jeopardy! format.

  \item \textbf{LAMBADA} \parencite{paperno2016lambada} (0-shot, 5,153 examples):
  Final-word prediction requiring full narrative context.

  \item \textbf{OpenBookQA} \parencite{mihaylov2018openbookqa} (0-shot, 500 examples):
  Multi-step reasoning and commonsense QA.

  \item \textbf{PIQA} \parencite{bisk2020piqa} (10-shot, 1,838 examples):
  Binary physical commonsense reasoning.

  \item \textbf{SQuAD} \parencite{rajpurkar2016squad} (10-shot, 10,570 examples):
  Extractive QA from Wikipedia passages.

  \item \textbf{Winograd Schema Challenge} \parencite{levesque2012winograd} (0-shot, 273 examples):
  Binary pronoun resolution requiring commonsense.

  \item \textbf{Winogrande} \parencite{sakaguchi2020winogrande} (0-shot, 1,267 examples):
  Adversarial extension of the Winograd Schema Challenge.

\end{itemize}
